# Toga - FoodMarket

Proyecto de la Escuela de JavaScript - Platzi 2019.

FoodMarket es un espacio para gustosos por la gastronomía pero que no tienen tiempo para comprar los productos para sus preparaciones.

Instalación

```
npm install
```

ejecución

```
npm run start
```
